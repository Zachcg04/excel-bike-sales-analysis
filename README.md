# Excel Bike Sales Dashboard  
*by Zachary Garrett*  
*Date: 11/2025*  

## Project Overview
This project demonstrates the full data cleaning and dashboard creation process in Excel, using a bike sales dataset combined with customer demographics. The dashboard allows for exploration of sales trends, customer behavior, and regional performance, providing actionable insights for business decision-making.

## Key Features
- Data cleaning: handled missing values, standardized formats, and removed duplicates  
- Pivot tables and charts for summarizing sales by region, product type, and customer demographics  
- Interactive Excel dashboard with slicers for filtering by month, product category, and region  
- Visualizations include bar charts, line charts, and heat maps to highlight sales trends  

## How to Use
1. Open the Excel workbook (`Excel Project Dataset (1).xlsx`)  
2. Use the dashboard sheet to explore sales trends by category, region, or customer demographics  
3. Use slicers to filter the data dynamically for specific analysis needs
